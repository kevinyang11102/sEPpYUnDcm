## 项目概述

基于Python开发的药品名称识别系统，主要用于用户输入文本中药品名称的提取与检索。系统具备完善的功能模块，实现了前台用户操作与后台管理。

## 技术栈

- Python
- MySQL

## 功能模块

### 前台功能

1. 匿名浏览
2. 用户注册与登录
   - 用户需注册并登录才能使用系统完整功能。
3. 个人中心
   - 查看个人信息、历史评论和查询历史。
4. 系统信息查看
5. 文本提取
   - 用户输入文本进行药品名称提取。
6. 文本搜索
   - 提供分类检索功能。

### 后台功能

1. 用户个人信息管理
2. 文本管理
3. 管理员信息管理
4. 用户信息管理
   - 管理员对用户信息进行实时管理。

## 系统角色

- 用户：匿名浏览、注册、登录、使用功能。
- 管理员：后台管理、用户信息与文本管理。
